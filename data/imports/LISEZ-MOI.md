# Dossier d'import documentaire

Deposez ici les documents qui doivent nourrir le projet.

## Fonctionnement

- Les fichiers poses dans ce dossier apparaissent automatiquement dans le centre documentaire.
- Les formats suivants sont lisibles directement dans l'application :
  - `.md`
  - `.txt`
  - `.json`
  - `.csv`
  - `.log`
  - `.pdf`
  - `.docx`

## Usage simple

1. Copiez vos documents dans `prototype/data/imports`
2. Ouvrez `Documents` dans l'application
3. Rechargez la page si besoin

## Important

- En local, cela marche tout de suite dans votre projet.
- Sur le site Vercel, les nouveaux fichiers importes doivent etre pousses sur GitHub puis redeployes pour apparaitre.
